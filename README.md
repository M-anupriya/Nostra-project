Nostra is a responsive e-commerce website developed using HTML, CSS, and JavaScript. The project focuses on building a modern online shopping interface with smooth navigation, responsive design, and interactive product browsing features such as search and filtering.

The website is designed to provide a simple and user-friendly shopping experience where users can explore product collections, search for specific items, and filter products easily.

🚀 Features

Responsive Design – Works smoothly on desktop, tablet, and mobile devices.

Home Page – Displays the main banner, navigation menu, and featured sections.

Collections Page – Shows different products available in the store.

Search Functionality – Users can search for products instantly using JavaScript.

Filter Functionality – Products can be filtered based on categories.

Contact Page – Allows users to send messages or inquiries.

Clean UI – Simple and modern layout for better user experience.
